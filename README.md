# Lakshya Jeet 🚀

Lakshya Jeet is a Flutter-based productivity and self-improvement app designed to help users track their daily growth.

## Features

* 📚 Study Tracker
* 💻 Tech Tracker
* 😴 Sleep Tracking
* ⚡ XP & Level System
* 🔥 Daily Streaks
* 🏆 Achievements
* 🎯 Daily Score Calculation
* 💾 Local Data Persistence with Hive

## Built With

* Flutter
* Dart
* Hive Database
